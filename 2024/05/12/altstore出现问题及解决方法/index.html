altstore出现问题及解决方法

发表于2024-05-12|更新于2024-05-12
|阅读量:

出现问题

altstore:This device has reached the maximum number of installed apps using a free developer profile: {(
“LBZSK8G7F9.com.LBZSK8G7F9.com.rileytestut.AltStore”,
“44W9SYPTC5.net.kdt.pojavlauncher.44W9SYPTC5”,
“44W9SYPTC5.com.44W9SYPTC5.com.rileytestut.AltStore”
)}

Please deactivate a sideloaded app in order to install another one.

If you’re running iOS 13.5 or later, make sure “Offload Unused Apps” is disabled in Settings > iTunes & App Stores, then install or delete all offloaded apps.

解决方式

第一种情况

设备开启了自动卸载未使用的app
在设置>App Store种关闭

第二种情况

设备侧载安装了过多app
卸载多余的侧载app

其他解决方法：安装troll store
