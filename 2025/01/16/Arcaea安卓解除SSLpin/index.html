Arcaea安卓解除SSLpin

发表于2025-01-16|更新于2025-01-16
|阅读量:

有不懂的？

可以使用arcaea@nekonekoniko120.com联系我

逆向工具

本次逆向使用ida
Arcaea 6.1.0

记得安装nop插件
文件在Arcaea的apk/lib/arm64-v8a/libcocos2dcpp.so

开始解除SSLpin

在strings里搜vtvtvt

点进去点avtvtvt

然后xref选第一个

按tab进入伪代码
找到这个结构

把标黄的部分nop掉（Edit-plugins-NOPPlugin）
然后拉到最底下找到这个

把标黄的部分nop掉

保存

点Edit-patch program-apply patches to input file
然后把文件扔回去签名对齐
恭喜🎉，SSLpin解除完成，可以使用Charles抓包来游玩私服了
